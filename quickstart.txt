ARCS - Quick Start Guide
=======================

This guide walks through bringing ARCS (Amateur Radio Call Search) online
using Docker Compose.

Prerequisites
-------------
- Linux host
- Docker and Docker Compose installed
- Internet access for FCC ULS data downloads

Starting ARCS
-------------
From the project directory:

  docker compose up -d

Once running:
- Web UI: http://<host>:8081
- XML API: http://<host>:8080/xml.php

Testing
-------
Callsign lookup:

  curl "http://127.0.0.1:8080/xml.php?callsign=W1AW"

Advanced search:

  curl "http://127.0.0.1:8080/xml.php?action=search&state=NE&city=Omaha"
